# Test Images

This directory contains sample images for testing the Firebase Converter function API.

## Usage

Place your test images here and reference them in the Bruno environment variables.

### Examples:

- `edt-test.png` - Sample schedule/timetable image for converter testing

You can use these images by:
1. Storing them locally in this directory
2. Referencing them via URL in the environment variables
3. Using the pre-request script to convert them to base64 format
